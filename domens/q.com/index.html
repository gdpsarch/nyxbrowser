📖 About Q.com

What is Q.com?

Q.com is a demo domain for Nyx Browser. It showcases how local HTML sites work with the site:// protocol and hash-based navigation.

📂 Folder Structure: 
domens/
└── q.com/
    └── index.html    (this file)

To add pages, create separate .html files or use hash navigation.
